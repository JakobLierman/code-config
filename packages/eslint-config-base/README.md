# eslint-config-jakoblierman-base

Personal ESLint base configuration.

## Installation

```bash
npm install --save-dev eslint-config-jakoblierman-base
```

```bash
yarn add --dev eslint-config-jakoblierman-base
```

## Usage

`.eslintrc`

```json
{
  "extends": "jakoblierman-base"
}
```
